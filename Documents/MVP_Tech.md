# MVP Technical Plan

## Goal
Launch a working system with authentication, upload, search, and download features.

---

## Tech Stack

### Frontend
- Next.js
- Tailwind CSS

### Backend
- Firebase Functions

### Database
- Firebase Firestore

### Auth
- Firebase Authentication

### Storage
- Cloudinary

---

## MVP Features

### Authentication
- Signup/Login
- Role-based access

---

### Admin Panel
- View users
- Ban/unban users
- Assign roles

---

### Faculty Panel
- Upload PDF
- Add metadata:
  - Title
  - Class
  - Subject
- Edit/Delete resources

---

### Student Panel
- Search resources
- Filters:
  - Class
  - Subject
- Download PDFs

---

## Non-MVP Features
- AI recommendations
- Notifications
- Analytics dashboard
